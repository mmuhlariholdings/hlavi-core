## Getting Started

A quick guide on how you can get started running and working on the applicatoin on your local machine.

### Requirements

- Rust 1.75 or higher
- Cargo

### Clone

```bash
git clone https://github.com/mmuhlariholdings/hlavi-core.git
cd hlavi-core
```

### Build

```bash
cargo build
```

### Test

```bash
cargo test
```
